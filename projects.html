Now Trivia!
This was a personal project I did in the Summer of 2016. The web-app is a trivia game based on the popular music album series Now That's What I Call Music! Players are given a song and must guess what album from the series it is from. The game is written in python using the Djano framework and is deployed through Heroku. Upcoming updates include Spotify API integration and new frontend design so stay tuned!
