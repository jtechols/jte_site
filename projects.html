Pac-Man AI
This was a class project for Computer Science 188 Artificial Intelligencce. I created the Pac-Man agent you see above in Python using a mulit-agent search algorithm. We also did a similar project using reinforcement learning.
Bear Maps
This was a class project for Computer Science 61B Data Structures. We created a mapping application that allows you to find the shortest path from two places in Berkeley. The application implements the A* Search algorithm to find the shortest distance and is written in Java.

Arcade Machine
This was a personal project I did with a Mechanical Engieering friend of mine. We modeled the cabinet using AutoDesk Fusion 360 and then laser cut the wood. The inside is comprised of an old PC monitor, a raspberry pi, and a USB microcontroller whose pins connect to all the buttons. The software for the Raspberry Pi is the open source project Retropie.
